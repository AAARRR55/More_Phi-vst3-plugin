## 2. Version History

### Current Versions

#### SnappySnap LE (Free Edition)
- **Cost**: Free
- **Limitations**:
  - 2 snapshots maximum
  - 1D fader morphing only (no 2D XY-pad)
  - No macro knobs
  - No breeding mode
  - No preset banks
  - No MIDI triggers

#### SnappySnap Full Version (Paid)
- **Features**:
  - 12 snapshots
  - 1D fader + 2D XY-pad morphing
  - 8 macro knobs
  - Breeding mode (genetic algorithm preset combination)
  - 16 banks x 128 presets
  - MIDI triggers for all controls
  - Drift mode (automatic parameter movement)
  - Link mode (synchronize controls)
  - Sanity Mode (excludes volume/bypass/mute from morphing)
  - AI Bridge with MCP (Model Context Protocol) integration

### Development Status
- **Active development**: The plugin is currently maintained and updated
- **MCP Integration**: Native AI agent integration supports Claude Desktop
- **Format**: Universal Binary supporting both Apple Silicon (arm64) and Intel (x86_64) natively

## 5. Unique Features

### AI Integration
- Native MCP (Model Context Protocol) support
- Direct integration with Claude Desktop for AI-controlled parameter manipulation
- Enables AI agents to control plugin parameters programmatically

### Parameter Interpolation
- Morphs between plugin parameters (not audio)
- 1D fader for linear interpolation
- 2D XY-pad for quad-point morphing
- Sanity Mode prevents critical parameters (volume, bypass, mute) from being affected

### Genetic Algorithm
- Breeding mode combines two parent snapshots to create new child sounds
- Enables exploration of new sonic possibilities through algorithmic combination

### Performance Features
- Drift mode for automatic parameter movement
- Link mode to synchronize multiple controls
- MIDI-triggerable parameters for live performance
