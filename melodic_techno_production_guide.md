## 1. PROJECT SETUP & FUNDAMENTALS

### Tempo & Key

| Parameter | Setting | Notes |
|-----------|---------|-------|
| **BPM** | 122-125 BPM | Sweet spot is 122-123 BPM |
| **Key** | A minor (beginners) | D minor, F minor, G minor also popular |
| **Time Signature** | 4/4 | Standard for the genre |

### Track Organization (Color-Coded)

| Track Group | Color | Contents |
|-------------|-------|----------|
| **Drums** | 🔴 Red | Kick, hi-hats, percussion, claps |
| **Bass** | 🔵 Blue | Sub-bass, mid-bass, bassline harmonics |
| **Melodic** | 🟢 Green | Pads, leads, arpeggios, plucks |
| **Atmosphere** | 🟣 Purple | Textures, field recordings, vocals |
| **FX Returns** | 🟡 Yellow | Reverb sends, delay sends, sidechain bus |

### Essential Effect Sends (Set Up Before Starting)

1. **Send A — Main Reverb**: Valhalla VintageVerb, Decay 2.8s, Pre-delay 45ms, High-cut 7.5kHz
2. **Send B — Delay**: Dotted 1/8th note (~367ms @ 122 BPM), 22% feedback, High-cut 4kHz
3. **Send C — Sidechain Bus**: Compressor triggered by kick, 6:1 ratio, 0.5ms attack, 150ms release

## 2. KICK DRUM - THE FOUNDATION

### Frequency Structure

| Layer | Frequency Range | Purpose |
|-------|-----------------|---------|
| Sub Layer | 50-80 Hz | Weight and foundation |
| Body Layer | 80-200 Hz | Punch and thickness |
| Transient/Click | 2-4 kHz | Definition and cut |

### Complete Kick EQ Settings

| Frequency | Action | Amount | Purpose |
|-----------|--------|--------|---------|
| 30-42 Hz | High-pass | 18dB slope | Remove sub rumble |
| 150 Hz | Cut | -1 dB | Make room for bass |
| 300 Hz | Cut | -3.5 dB | Remove boxiness |
| 1.5-4 kHz | Boost | +6 dB | Add punch/click |
| 16 kHz | Low-pass | - | Remove harshness |

### Compression Settings

| Parameter | Value |
|-----------|-------|
| Ratio | 3:1 to 4:1 |
| Attack | 3-5 ms |
| Release | 100-200 ms |
| Gain Reduction | 2-4 dB |

### Kick Layering Technique

**Layer 1 (Sub)**: Pure low-end kick (50-80Hz fundamental)
- High-pass at 30Hz
- Low-pass at 100Hz
- Keep in mono

**Layer 2 (Body)**: Mid-range punch kick
- High-pass at 80Hz
- Low-pass at 3kHz
- Adds thickness

**Layer 3 (Click)**: High transient for definition
- High-pass at 2kHz
- Adds attack and cut

**Group Processing**:
- Mono below 100Hz
- Gentle saturation/analog warmth
- Light compression (2:1)

## 3. BASS DESIGN & PROCESSING

### Bass Layering Strategy

| Layer | Type | Frequency | Role |
|-------|------|-----------|------|
| Sub Bass | Sine wave | 40-80 Hz | Foundation, mono only |
| Mid Bass | Saw/Square | 80-200 Hz | Character and warmth |
| High Bass | Filtered saw | 120-500 Hz | Edge and presence |

### Bass Sound Design (Serum/Diva Template)

```
Oscillator A: Saw wave
- 2-3 unison voices
- Detune: 0.05-0.15 semitones
- Creates harmonic richness

Oscillator B: Square wave
- 1 octave lower
- Mix: 30-40%
- Adds sub-harmonic weight

Filter: Low-pass 24dB
- Cutoff: 200-800 Hz
- Slope: 24dB

Filter Envelope:
- Attack: 0ms
- Decay: 150-300ms
- Sustain: 40-60%
- Release: 200-400ms

LFO Modulation:
- Target: Filter cutoff
- Shape: Sine or Triangle
- Rate: 1/8th note
- Depth: 15-25%
```

### Bass EQ Settings

| Frequency | Action | Amount | Purpose |
|-----------|--------|--------|---------|
| 25-30 Hz | High-pass | - | Clean sub |
| 60-80 Hz | Boost | +1-2 dB | Add weight (if thin) |
| 200-250 Hz | Cut | -2-3 dB | Prevent muddiness |
| 300-400 Hz | Cut | -1-3 dB | Remove boxiness |

### Bass Processing Chain

```
1. EQ (surgical cuts at 200-250Hz)
2. Compression (4:1, 10-30ms attack, 100-200ms release)
3. Saturation (analog-style, gentle)
4. Sidechain compression (from kick)
5. Mono maker (below 120Hz)
```

### Bass Pattern Guidelines

- Root notes (A) on strong beats: 1, 2, 3, 4
- Fifths (E) on off-beat 8th positions: 1.5, 2.5, 3.5, 4.5
- Ghost notes at lower velocity between 8th notes

**Velocity Tiers:**
- Strong (100-110): On-beat root notes
- Medium (75-85): Off-beat 8th notes
- Subtle (50-65): 16th note fills

## 5. MIXING CHAIN PROCESSING

### Standard Signal Chain by Element

**KICK:**
```
EQ (surgical) → Compression → Saturation (subtle) → Limiter (gentle)
```

**BASS:**
```
EQ → Compression → Saturation → Sidechain (from kick) → Mono maker (<120Hz)
```

**PADS:**
```
EQ (high-pass) → Chorus (15-25%) → Compression (2:1) → Reverb Send → Sidechain
```

**LEADS:**
```
EQ → Compression → Saturation → Delay (dotted 1/8) → Reverb → Sidechain
```

**ARPEGGIOS:**
```
EQ → Compression → Delay (1/8 dotted) → Reverb → Sidechain (6:1)
```

**ATMOSPHERIC TEXTURES:**
```
EQ (high-pass at 60-80Hz) → Reverb → Sidechain (2:1, gentle)
```

### Master Bus Chain

```
1. Pro-Q 3
   - Gentle low roll-off < 25 Hz
   - Gentle high shelf

2. Saturn 2
   - Mid band drive
   - Mix ~20%

3. Bus Glue Compressor
   - Ratio: 2:1
   - Slow attack
   - Auto release

4. Limiter (Pro-L 2 / Ozone Maximizer)
   - Ceiling: -0.8 dB
   - Clean limiting

5. Metering (SPAN / Youlean)
   - Target: -12 to -10 LUFS integrated
```

## 7. REVERB & DELAY SETTINGS

### Main Reverb (Valhalla VintageVerb)

```
Algorithm: 1970s Hall
Decay: 2.2-2.8 seconds
Pre-delay: 45ms
High-cut: 7.5 kHz
Mix: 15-25%
```

### Plate Reverb (Alternative)

```
Algorithm: Plate
Decay: 1.8-2.5 seconds
Pre-delay: 25-40ms
High-frequency damping: Gentle
Mix: 15-25%
```

### Delay Settings

| Type | Timing @ 122 BPM | Feedback | Use Case |
|------|------------------|----------|----------|
| Dotted 1/8 | 367 ms | 22% | Hypnotic echoes, main delay |
| 1/8 Note | 246 ms | 15-25% | Lead fills, rhythmic echoes |
| 1/4 Note | 492 ms | 20% | Atmospheric, sparse |
| 3/16 Note | 184 ms | 25-35% | Rhythmic variation, groove shift |

### Breakdown Reverb Technique

**Automation Strategy:**
- Increase reverb send by +3 to +6 dB when kick drops out
- Automate size/modulation depth over 16 bars
- Creates "expanding into infinite space" effect
- Return to normal levels for drop

### Shimmer Reverb (For Pads/Breakdowns)

```
Plugin: Valhalla Shimmer
Size: 80-90%
Pitch shift: +12 semitones (or +7 for fifth)
Feedback: 60-80%
Mix: 20-30%
Modulation: Slow (0.1-0.3 Hz)
```

## 9. SOUND DESIGN BLUEPRINTS

### Evolving Pad Chain (Breakdowns/Intros)

```
Pigments (saw wave, 7 unison, slow attack)
    ↓
Pro-Q 3 (HP @ 150 Hz, surgical cuts at 200-250Hz)
    ↓
Saturn 2 (Warm Tape, Drive 10-20%, Mix 25-35%)
    ↓
Valhalla Shimmer (Mix 20-30%, Size 80-90%)
    ↓
LFO Tool (gentle duck to kick, 20-30% reduction)
```

**Automation:**
- 16-bar sweep on cutoff (500 Hz → 2 kHz)
- +3-6 dB reverb mix lift into transitions
- Light width increase in breakdowns

### Melodic Lead Chain (Main Hook)

```
Diva (saw/square, 3-4 unison, slight detune)
    ↓
EchoBoy (Dual Echo, 1/8 or 3/16, FB 25-35%, HP 200Hz)
    ↓
Pro-Q 3 (surgical, preserve 1-5kHz)
    ↓
Decapitator (Low drive, parallel blend)
    ↓
Valhalla VintageVerb (Hall, 2s decay, 20% mix)
```

**Automation:**
- Cutoff envelope amount + delay feedback
- High-shelf -1 dB in busy sections
- Portamento 50-200ms on 4th/5th intervals

### Driving Bass Chain (Groove Section)

```
Hive 2 / Diva (mono, saw into LP filter)
    ↓
Saturn 2 (Tube, very low drive, Mix 20%)
    ↓
Pro-Q 3 (HP @ 30Hz, cut @ 200-250Hz)
    ↓
LFO Tool (Sidechain, 4:1 ratio, 150ms release)
    ↓
Utility (Mono below 120Hz)
```

**Automation:**
- Subtle drive variation every 8 bars
- Filter envelope depth tweaks
- LFO rate modulation for movement

### Atmospheric Texture Chain

```
Granular synthesis (acoustic source)
    ↓
Heavy time-stretch (200-400%)
    ↓
Pro-Q 3 (HP @ 60-80Hz, LP @ 2-3kHz)
    ↓
Valhalla VintageVerb (Hall, 3-5s decay, 100% wet)
    ↓
LFO Tool (Sidechain to kick, gentle)
    ↓
Mix at -20 to -30 dB
```

## 10. REFERENCE MIXING WORKFLOW

### Step-by-Step Mixing Process

**Step 1: Level the Kick**
- Play kick in solo
- Level around -7 or -6 dB
- Results in final track peaking at -4/-3 dB
- Provides adequate headroom for mastering

**Step 2: Solo Core Elements**
- Solo kick, hats, snare, bass
- Take in/out to check interaction
- Adjust low cuts as needed
- Drums, kick, and bass take the most time

**Step 3: Visual Gain Staging**
- Kick always slightly louder than bass
- Find right balance between kick and snare
- Prevent limiter "panic attacks" from combined peaks

**Step 4: Set Low Cuts**
- Kick: High-pass at 30-42 Hz
- Small punchy kick: 42 Hz
- Subby techno kick: 30 Hz
- A/B test - often sounds better without, but trust the cut

**Step 5: Mix Effects Before Synths**
- Adjust atmospheric effects group first
- Prevents FX from becoming excessively loud
- Balance white noise and texture layers

**Step 6: Follow the Arrangement**
- Play track from beginning
- Add elements step by step
- Stop and adjust as needed
- Mix evolves with the track structure

**Step 7: Mix As You Go**
- Don't overmix during production
- But make it sound good before "mixing stage"
- Add drum rooms and compressors during production

**Step 8: Create Space**
- Use analyzers to identify frequency buildup
- Check 2000 Hz area for conflicts
- Create space for each instrument

**Step 9: The "Miss It" Test**
- Disable elements one by one
- Ask: "Do you miss this element?"
- If not, remove it
- Less is more in melodic techno

**Step 10: Take Breaks**
- Finish track a week before deadline
- Return with fresh ears
- Identify issues not noticed before

### Reference Track Workflow

1. Load 2-3 reference tracks from Afterlife, Innervisions, or Diynamic
2. Volume-match references to your pre-master level
3. Compare frequency balance using spectrum analyzer
4. A/B test throughout entire mixing process
5. Check on multiple systems: monitors, headphones, phone speakers

## 11. COMMON PROBLEM AREAS

### Frequency Conflict Resolution

| Problem Frequency | Cause | Solution |
|-------------------|-------|----------|
| 200-250 Hz | Kick/bass mud, pad buildup | Cut -2 to -4 dB on pads/leads |
| 300-400 Hz | Boxiness in kicks/bass | Surgical cut, high Q |
| 400-600 Hz | General mud, dense mix | Surgical cuts, remove unnecessary content |
| 1-5 kHz | Harshness, ear sensitivity | Dynamic EQ, gentle cuts |
| 6-9 kHz | Hi-hat harshness | Notch cut, 909 kits often problematic |

### Common Beginner Mistakes

**1. Over-Compression**
- Avoid aggressive ratios (8:1-12:1)
- Use 2:1-4:1 ratios
- Preserve dynamics and breathing

**2. Muddy Low-Mids (200-500Hz)**
- Most common mixing problem
- Apply surgical EQ cuts
- Use spectrum analyzer to identify overlap

**3. Static Arrangements**
- Create energy valleys at 25%, 50%, 75%
- Use filter automation, reverb sends
- Constant subtle evolution is key

**4. Revealing Everything Too Early**
- Save best melodic content for breakdown
- Intro is appetizer, not main course
- Create emotional journey

**5. Wide Bass Frequencies**
- Keep <120Hz in mono
- Check mono compatibility
- Use correlation meter

**6. Ignoring Reference Tracks**
- Always use 2-3 professional references
- Volume-match before comparing
- A/B test throughout process

### Phase Alignment Check

**Detection:**
- Listen in mono - if bass disappears, phase issues exist
- Use oscilloscope/meter to visualize
- Peaks and troughs should align

**Solutions:**
- Adjust waveform starting points in synths
- Use phase alignment plugins (Voxengo PHA-979, Auto Align)
- Nudge samples by milliseconds
- Flip phase on one layer

### Resonant Frequency Hotspots

| Range | Common Issues | Action |
|-------|---------------|--------|
| 200-800 Hz | Muddy low-mids | Check all melodic elements |
| 1-5 kHz | Human ear sensitivity | Dynamic EQ treatment |
| 6-9 kHz | Hi-hat harshness | Notch cut if problematic |
| 3000 Hz | Vocal harshness | Experiment with cuts |

## ARTIST-SPECIFIC TECHNIQUES

### Stephan Bodzin-Style Approach
- **Philosophy:** Fewer layers, deeper automation
- **Tools:** Diva (mono bass/lead), analog-style filters
- **Technique:** Slow macro moves over 16-32 bars
- **Key:** D minor for heavy, emotional tracks

### Tale Of Us / Anyma-Style Atmospheres
- **Philosophy:** Cinematics and space as composition
- **Tools:** Pigments/Hive for pads + Valhalla/Eventide chains
- **Technique:** Width automation across sections
- **Key:** G minor for balance of darkness and uplift

### ARTBAT-Style Arpeggios
- **Technique:** A natural minor → A Dorian (raise F to F#) during build-up
- **Effect:** Single note change creates emotional ascent
- **Processing:** Heavy sidechain, dotted delays

### Miss Monique-Style Energy
- **Tempo:** 122 BPM sweet spot
- **Approach:** Hypnotic repetition with micro-variations
- **Bass:** Rolling 16th note patterns with velocity tiers
- **Arrangement:** Extended builds, subtle progression

## FINAL CHECKLIST

### Before Exporting

- [ ] Mix peaks at -6 dB (pre-master)
- [ ] All sub-bass (<120Hz) is mono
- [ ] Kick and bass are phase-aligned
- [ ] Sidechain compression on bass, pads, leads
- [ ] High-pass filters on all non-bass elements
- [ ] Checked in mono for phase issues
- [ ] A/B tested against 2-3 reference tracks
- [ ] Listened on multiple systems
- [ ] Taken breaks, returned with fresh ears
- [ ] No clipping on master bus
- [ ] Reverb sends automated for breakdowns
- [ ] No harsh resonances above 6 kHz

### Master Bus Targets

- **Peak Level:** -4 to -3 dB (pre-master)
- **LUFS Integrated:** -12 to -10 LUFS
- **True Peak:** -1.0 dBTP (maximum)
